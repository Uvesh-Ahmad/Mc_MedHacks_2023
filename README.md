## About McMedHacks

[McMedHacks](https://mcmedhacks.com/) is a first-of-its-kind workshop series designed to provide hands-on experience 🧑‍💻 with **Deep Learning in Medical Imaging** using the `Python` programming language.

It is an initiative by the **McGill community** aimed at democratizing medical imaging through Deep Learning and bringing together individuals from diverse backgrounds to solve real-world challenges using **Artificial Intelligence 🤖 in medical image analysis**.
